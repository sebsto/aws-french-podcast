description: "Dans cet épisode, Seb vous emmène au coeur des annonces de l'AWS Summit New York 2026. Au programme : Amazon S3 Annotations qui permet d'attacher jusqu'à 1 Go de métadonnées structurées et interrogeables directement à vos objets S3, une fonctionnalité qui change la donne pour les agents IA. AWS WAF inaugure la monétisation du trafic IA - les éditeurs de contenu peuvent désormais facturer les bots IA qui accèdent à leurs sites. Côté IA, Amazon Bedrock Managed Knowledge Base passe en disponibilité générale avec Smart Parsing et un Agentic Retriever, tandis que Web Search arrive sur Bedrock AgentCore pour ancrer vos agents dans des données web actuelles. Les développeurs ne sont pas en reste avec AWS Blocks, un nouveau framework TypeScript open-source pour construire des backends sur AWS sans toucher à l'infrastructure, et AWS DevOps Agent qui ajoute la gestion des releases. Enfin, S3 Vectors baisse ses tarifs de requêtes de 80% et Amazon Quick se dote d'agents autonomes."
