=== DESCRIPTION ===
Épisode 372, enregistré après le Summit AWS de New York. Cette édition est dense en annonces, particulièrement autour de l'écosystème agentique.

On commence avec S3 Annotations, une nouvelle fonctionnalité qui permet d'attacher jusqu'à 1 Go de métadonnées structurées par objet, interrogeables directement via Athena. S3 Vectors baisse aussi ses prix jusqu'à 80% sur les gros index et supporte désormais 10 000 résultats par requête.

Côté Bedrock, la grande nouveauté est Managed Knowledge Base, une option zero-config qui se distingue de l'approche existante où il fallait choisir sa base vectorielle et configurer le chunking. Six connecteurs natifs sont disponibles, dont Sharepoint, Confluence et Google Drive. Agent Core gagne un outil de web search intégré, compatible MCP.

Un développement intéressant à suivre : WAF Bot Control peut maintenant facturer les bots IA pour l'accès au contenu via le protocole HTTP 402, pendant qu'Agent Core Payment permet aux agents de payer. Les deux côtés de l'équation sont couverts.

Autres annonces : AWS Security Agent fait du threat modeling avec la méthodologie STRIDE, le Dev Agent gère maintenant les releases, et AWS Blocks arrive en preview pour les développeurs full-stack TypeScript. Gemini 2.5 de Google et Grok de xAI sont disponibles sur Bedrock. Amazon Q gagne des agents autonomes et de nouveaux connecteurs.

La semaine prochaine, interview de Julie Paget du Groupe SEB sur leur usage d'Amazon Q.
