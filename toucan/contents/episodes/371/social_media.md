le podcast 🎙 AWS ☁️ en 🇫🇷

Spécial AWS Summit New York 2026 ! Cette semaine, un déluge d'annonces.

Le gros morceau : Amazon S3 Annotations. Vous pouvez désormais attacher jusqu'à 1 Go de métadonnées structurées et interrogeables directement sur vos objets S3. Un game-changer pour les agents IA qui ont besoin de contexte sur les données qu'ils manipulent.

Autre annonce surprenante : AWS WAF permet maintenant aux éditeurs de contenu de facturer les bots IA qui accèdent à leurs sites. La monétisation du trafic IA, c'est nouveau.

Également au programme :
→ Bedrock Managed Knowledge Base en GA avec Smart Parsing
→ Web Search sur Bedrock AgentCore
→ AWS Blocks, un framework TypeScript open-source pour les backends
→ S3 Vectors baisse ses tarifs de 80%
→ Amazon Quick avec des agents autonomes
