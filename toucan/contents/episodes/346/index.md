title: "13 février 2026 - Kiro 0.9, Opus 4.6, Bedrock, EC2, Aurora DSQL, et ECS"
description: "Dans cet épisode du podcast AWS en français, Kiro est à l'honneur avec la version 0.9 qui apporte les subagents personnalisés, les skills et de nouveaux contrôles pour les entreprises. Kiro adopte aussi le protocole ACP pour s'intégrer dans d'autres IDE comme DataGrip ou Emacs, et accueille Claude Opus 4.6, le nouveau modèle phare d'Anthropic, ainsi que des modèles open weight comme DeepSeek v3.2, MiniMax 2.1 et Qwen3 Coder Next. Côté AWS, Amazon Bedrock propose désormais les structured outputs pour des réponses JSON conformes à vos schémas, et AgentCore Browser supporte les profils de navigateur pour réutiliser les sessions authentifiées. AWS Builder ID permet maintenant de se connecter avec son compte Apple. On parle aussi des nouvelles instances EC2 C8id, M8id et R8id avec processeurs Intel Xeon 6, des déploiements linear et canary pour Amazon ECS avec les Network Load Balancers, de l'expansion d'Aurora DSQL dans de nouvelles régions, de la réduction de prix d'AWS Network Firewall, et de la nouvelle GitHub Action pour Elastic Beanstalk."
