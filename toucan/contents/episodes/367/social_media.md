Cette semaine, un épisode dense avec les annonces des deux dernières semaines !

Claude Platform passe en GA sur AWS : l'expérience native d'Anthropic est désormais accessible directement depuis votre compte AWS.

Au programme également :

→ AWS Interconnect : connectivité multicloud avec Oracle Cloud Infrastructure (preview)

→ Aurora DSQL supporte le change data capture (preview)
→ AWS Transform débarque dans Kiro et Claude
→ Pause hooks pour les déploiements ECS
→ mTLS passthrough et OCSP sur CloudFront
→ Nouveau comportement de retry par défaut des SDKs AWS (à comprendre avant novembre 2026)
