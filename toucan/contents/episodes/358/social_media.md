le podcast 🎙 AWS ☁️ en 🇫🇷 

La nouveauté de la semaine : Amazon S3 Files. Vos buckets S3 deviennent accessibles comme des systèmes de fichiers NFS grâce à une couche EFS qui sert de cache intelligent devant S3. Les données restent dans S3 comme source de vérité, mais vos applications peuvent faire ls, cp, et utiliser les sémantiques de locking POSIX. Latence d'une milliseconde. Un vrai changement pour les workloads legacy et les agents IA qui accèdent à vos données en lisant simplement un filesystem.

Également au programme :
→ Polly passe au streaming bidirectionnel HTTP/2 pour les intégrations LLM temps réel
→ La console de durabilité gagne son indépendance avec accès API et rapports Scope 1-2-3
→ ECS Managed Instances supporte les daemons pour séparer ops et applicatif
→ App Runner, WorkMail, WorkSpaces Thin Client et d'autres entrent en fin de vie
