description: "Dans cet épisode du podcast AWS en français, Seb fait le tour des annonces des deux dernières semaines. Anthropic Claude Opus 4.7 arrive dans Amazon Bedrock, avec des gains sur le coding agentique (87,6% sur SWE-bench Verified) et une fenêtre de contexte d'1M de tokens. AWS Interconnect passe en disponibilité générale avec deux modes : multicloud pour connecter votre VPC à Google Cloud et Azure via le backbone AWS, et last-mile pour simplifier la connectivité privée depuis vos data centers. Amazon Bedrock AgentCore ajoute un managed harness en preview, une CLI, et des skills pour assistants de code. Côté Lambda, les durable functions arrivent dans 16 régions supplémentaires, le SDK Java passe en GA, et les fonctions peuvent désormais monter des buckets S3 comme des systèmes de fichiers via S3 Files. AWS Secrets Manager supporte le post-quantum TLS avec ML-KEM et étend les managed external secrets à MongoDB Atlas et Confluent Cloud. Également au programme : AWS Transform dans Kiro et VS Code, GPU auto-repair pour ECS Managed Instances, resource visibility settings sur EC2, referrers dans ECR Pull Through Cache, connecteur PHP pour Aurora DSQL, et les mises à jour trimestrielles d'Amazon Corretto."
