le podcast 🎙 AWS ☁️ en 🇫🇷

Capital Fund Management (CFM), gestionnaire de fonds quantitatif, a migré sa plateforme de calcul vers AWS. Julien Lafaye (Directeur) et Otmane El Ansary (Associate) nous racontent comment CFM Graph orchestre des graphes d'exécution Python sur AWS Batch, jusqu'à 100 000 vCPU et 80 TB de données en output.

→ Architecture : API Gateway, Lambda, DynamoDB, AWS Batch
→ 80-90% d'instances Spot pour la recherche
→ Défis : quotas, rate limiting S3, allocation des coûts
→ Approche hybride on-prem / cloud pendant la migration
