le podcast 🎙 AWS ☁️ en 🇫🇷

Cette semaine, les agents IA gagnent du terrain sur AWS.

L'AWS MCP Server passe en GA : trois outils (call_aws, search_documentation, run_script) donnent aux agents un accès authentifié aux 15 000 APIs AWS sans exploser la fenêtre de contexte. Il rejoint le nouveau Agent Toolkit for AWS qui regroupe plus de 40 skills validées et trois plugins prêts à l'emploi.

Autres annonces de la semaine :

→ Amazon WorkSpaces pour agents IA (preview) : un bureau virtuel pour piloter les apps desktop legacy sans API
→ Amazon Bedrock AgentCore Payments (preview) : les agents paient des APIs via le protocole x402 (Coinbase, Stripe)
→ AWS SAM supporte les WebSocket APIs d'API Gateway et BuildKit pour les conteneurs Lambda
→ Les quotas IAM doublent pour les rôles, policies et instance profiles
→ Console Mobile App : graphiques interactifs, résumés de logs par IA, recherche en langage naturel pour CloudWatch Alarms
