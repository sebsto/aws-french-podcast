title: "8 mai 2026 - AWS MCP Server GA, Agent Toolkit, WorkSpaces pour agents IA, AgentCore Payments, et SAM WebSocket"
description: "Dans cet épisode du podcast AWS en français, Seb fait le tour des annonces de la semaine autour des agents IA et du serverless. L'AWS MCP Server passe en disponibilité générale avec un ensemble compact d'outils (call_aws, search_documentation, run_script) qui donnent aux agents IA un accès authentifié aux 15 000 APIs AWS sans saturer la fenêtre de contexte. Il s'intègre au nouveau Agent Toolkit for AWS, qui rassemble le MCP Server, plus de 40 skills validées par les équipes de service, et trois plugins (AWS Core, AWS Data Analytics, AWS Agents). Amazon WorkSpaces donne en preview un bureau virtuel aux agents IA pour leur permettre de piloter les applications desktop legacy sans APIs, via MCP et avec des audit trails complets dans CloudTrail. Amazon Bedrock AgentCore ajoute Payments en preview, qui permet aux agents de payer autonomement des APIs et MCP servers grâce au protocole x402, en partenariat avec Coinbase et Stripe. Côté serverless, AWS SAM supporte désormais les WebSocket APIs d'API Gateway et BuildKit pour les images de conteneurs Lambda. Également au programme : les quotas IAM qui doublent pour les rôles, les policies et les instance profiles, et la Console Mobile App qui gagne des graphiques interactifs, des résumés de logs générés par IA et la recherche en langage naturel pour CloudWatch Alarms."
